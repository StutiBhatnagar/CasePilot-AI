# 🚀 CasePilot AI

An AI-powered MBA Case Study Analyzer built using **Google Gemini AI**.

CasePilot AI helps MBA students, consultants, and business professionals analyze business case studies within seconds by generating structured strategic insights.



## ✨ Features

- 📄 Executive Summary
- 🎯 Problem Statement
- 📊 SWOT Analysis
- 🌍 PESTLE Analysis
- 🏢 Porter's Five Forces
- 👥 Stakeholder Analysis
- 💰 Financial Impact
- ✅ Strategic Recommendations
- 📅 30-60-90 Day Implementation Plan
- 🤖 AI-Powered Business Analysis



## 🛠 Tech Stack

- Google Gemini AI
- Lovable
- React
- TypeScript
- Tailwind CSS
